# ubuntu-dev-box-with-ansible

## Requisiti
```
sudo apt install git ansible
ansible-galaxy collection install community.general
```

Git clone del progetto

## Start not dev
```
ansible-playbook site.yml
```

## Start dev
```
ansible-playbook site_dev.yml
```
